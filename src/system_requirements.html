System Requirements

Although Academy of Learning Career College Campuses are equipped with all the software and hardware required for you to sucessfully complete your courses, remote learning might be required depending on your personal circumstances.

Here are the system requirements to properly take advantage of our courses when learning remotely.

Basic Requirements (Most Students) Graphic Design, Web Developer, Web Design and Software Developer
Operating System Windows 10 or 11, Home or Pro
Memory(RAM) 4 GB or more 16 GB or more
Processor Any Intel® Core i5 or better, AMD® Ryzen 5 or better.
Internet Access 1 Mpbs+
Accessories
  • Webcam
  • Headset
  • External Full sized keyboard
  • External Mouse
  • Webcam
  • Headset
  • External Full sized keyboard
  • External Mouse
  • (Optional) Drawing Tablet or flatbed scanner
Display 14 inches or more 23 inches ore more. Most laptops support an external display if using a laptop
Software
  • No Software purchase required. All students will be given access to the software required for their courses.
  • We do not recommend purchasing an Office license until after graduation.
  • Our typing software will only function on Windows.

Please note that our Microsoft Office courses are based on the Windows Version of the software, so if you are running macOS, some of the instructions will not match.
Additionally, there is no macOS version of Microsoft Access. A virtual lab is available for this course at an additional cost.
